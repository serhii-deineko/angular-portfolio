## ğŸš€ Installation and Setup

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn
- Angular CLI

### Install Dependencies

```bash
npm install
# or
yarn install
```

### Development Server

```bash
npm start
# or
yarn start
```

The application will be available at `http://localhost:4200`

### Production Build

```bash
npm run build
# or
yarn build
```

### Run Tests

```bash
npm test
# or
yarn test
```
